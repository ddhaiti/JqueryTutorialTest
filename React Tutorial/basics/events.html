Events

Introduction

There is a set of event handlers that you can bind in a way that should look familiar to anybody who knows HTML. Some names of React event handlers are as follows:

  • Clipboard events
  • Composition events
  • Keyboard events
  • Focus events
  • Form events
  • Mouse events
  • Selection events
  • Touch events
  • UI events
  • Wheel events
  • Media events
  • Image events
  • Animation events
  • Transition events

They are writen in lowerCamelCase like "onClick", then we give them a function. 


function makeLake (e) {
  // Call this extremely useful function on an <img>.
  // The <img> will become a picture of a lake.
  e.target.setAttribute('src', '../assets/img/lake.png');
  e.target.setAttribute('alt', 'lake');
}

var forest = (
	<img
	src="../img/forest.png"
	alt="forest"
    	onClick = {makeLake}/>
);

ReactDOM.render(forest, document.getElementById('app'));
