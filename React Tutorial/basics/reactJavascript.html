React With Javascript

Introduction

To begin using React we need to be able to create ReactElements. These are our primary types in React, it is a virtual representation of a DOM element. We also have ReactText and they are both ReactNodes. An array of ReactNodes is a ReactFragment. To create a ReactElement we use the React.creatElement() function, our parameter will be type of element we want to make


var reactElement = React.createElement('h1');

What we did here is we created a react element 'reactElement'. These are the parameters of our function React.createElement(type, props, children);. The type is the tag in html, the props are the attributes suchs as class and style, and the children are the content of this tag, which can be other react elements. We have shown how to use type, here are the other parameters.


var reactElement = React.createElement('h1', { className: 'header' });
This is our props parameter. Notice when using class in Javascript we need to use 'className' instead, because 'class' is a reserved keyword in Javascript.

var reactElement = React.createElement('h1', { className: 'header' }, 'This is React');
This is our children parameter, but remember we can add more to this parameter

var h1 = React.createElement('h1', { className: 'header', key: 'header' }, 'This is React');
var p = React.createElement('p', { className: 'content', key: 'content' }, "And that's how it works.");
var reactFragment = [ h1, p ]; var section = React.createElement('section', { className: 'container' }, reactFragment);
Now this looks a bit tricky, but go through it line by line and you'll recognize everything in this block of code.
